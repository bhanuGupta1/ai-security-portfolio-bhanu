# Automation Frameworks

This folder contains test automation frameworks demonstrating different tools and approaches.

## Cypress Framework (Complete)
E2E test automation against [Sauce Demo](https://www.saucedemo.com) using Page Object Model, custom commands, and data-driven testing. 24 test cases with CI/CD.

**Folder:** [`cypress-framework/`](./cypress-framework)

## Playwright Framework (Planned)
Cross-browser E2E automation with TypeScript, visual regression testing, and mobile emulation.

## Selenium Java Framework (Planned)
Java-based automation using Maven, TestNG, and Page Object Model for enterprise-style testing.
